# Library Management System

## Description

A simple desktop application built with **Python** and **Tkinter** to manage library books through a graphical user interface.

## Features

* Add new books
* View all books
* Issue books
* Return issued books
* Simple and user-friendly interface

## Technologies Used

* Python
* Tkinter

## How to Run

1. Install Python.
2. Save the project files.
3. Open a terminal in the project folder.
4. Run:

   ```bash
   python Library_System.py
   ```

## Purpose

This project is designed for beginners to practice Python programming, functions, lists, dictionaries, loops, and GUI development using Tkinter.
